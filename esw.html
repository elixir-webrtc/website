Throwback to the first edition of Elixir Stream Week We had an amazing time hearing from top Elixir developers. Let's take a look back:
José Valim Dashbit - Creator of Elixir
What's new in Elixir 1.18
Elixir v1.18 is a massive step up in several directions. Watch the recording to learn more about the updates on the type system research, Language Server support, built-in JSON capabilities, and much more. Watch now
Mateusz Front Software Mansion - Creator of Membrane
Boombox explained
Boombox is a tool built on Membrane that makes it easier to kick off your multimedia journey. In this stream, Mateusz outlined the Boombox API and how to connect it with the OpenAI multimodal live streaming. Watch now
Chris McCord Fly.io - Creator of Phoenix
Live-coding real-time Twitch clone
It's possible to build a real-time Twitch clone in just one sitting - all while ensuring it's scalable and distributing it using LiveView, Elixir WebRTC and Fly.io. Want proof? Check the video! Watch now
Jonatan Kłosko Dashbit - Creator of Livebook
Q&A about Livebook, Elixir Nx and Bumblebee
Have some questions about Elixir ML Libraries? Chances are they have already been answered during our Q&A session. Take a comfy seat, grab a nice snack, and watch the recording! Watch now
Filipe Cabaço Software Engineer at Supabase
Real-time system using Elixir - Supabase stories
Learn what made Elixir the right choice for Supabase's Realtime. Delve into some of the implementation details regarding clustering, message sending, process registration, and more. Watch now
